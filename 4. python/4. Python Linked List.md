## Overview

A **linked list** is a linear data structure where each element (called a *node*) stores its own value and a pointer to the next node. Unlike Pythonâ€™s builtâ€‘in list (array), a linked list:

* Has dynamic size (nodes can be inserted/deleted in *O(1)* time once you reach the position).
* Does *not* support efficient random access (*O(n)* to reach index *i*).

## 1. Defining a Node

Each node holds data plus a `next` pointer.

```python
class Node:
    def __init__(self, data=None):
        self.data = data        # stored value
        self.next = None        # pointer to next Node
```

## 2. The LinkedList Wrapper

We keep a dummy `head` node to simplify edge cases.

## 3. Core Methods

### 3.1 `append(data)`

Add a new node to the *end*.

```python
def append(self, data):
    new = Node(data)
    cur = self.head
    # traverse to last node
    while cur.next:
        cur = cur.next
    cur.next = new
```

**Example**:

```python
>>> ll = LinkedList()
>>> ll.append(10)
>>> ll.append(20)
>>> ll.display()
[10, 20]
```

### 3.2 `length()`

Count nodes (excluding dummy head).

```python
def length(self):
    count, cur = 0, self.head
    while cur.next:
        count += 1
        cur = cur.next
    return count
```

**Example**:

```python
>>> ll.length()
2
```

### 3.3 `display()`

Return a Python `list` of stored values.

```python
def display(self):
    elems, cur = [], self.head
    while cur.next:
        cur = cur.next
        elems.append(cur.data)
    print(elems)
```

### 3.4 `get(index)`

Retrieve value by 0â€‘based `index`.

```python
def get(self, index):
    if index < 0 or index >= self.length():
        raise IndexError("Index out of range")
    cur, i = self.head.next, 0
    while i < index:
        cur = cur.next
        i += 1
    return cur.data
```

**Examples**:

```python
>>> ll.get(0)
10
>>> ll.get(1)
20
```

Error:

```python
>>> ll.get(2)
IndexError: Index out of range
```

### 3.5 `erase(index)`

Remove node at `index`.

```python
def erase(self, index):
    if index < 0 or index >= self.length():
        raise IndexError("Index out of range")
    prev, cur, i = self.head, self.head.next, 0
    while i < index:
        prev, cur = cur, cur.next
        i += 1
    prev.next = cur.next
```

**Example**:

```python
>>> ll.erase(0)
>>> ll.display()
[20]
```

## 5. Variations & Extensions

* **Singly vs. Doubly Linked List**: add `prev` pointer for *O(1)* removals.
* **Circular Linked List**: last node points back to head.
* **Sentinel/Dummy Nodes**: simplify edge cases at head/tail.
